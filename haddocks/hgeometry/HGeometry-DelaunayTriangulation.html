delaunayTriangulation :: (Foldable1 set, Point_ point 2 r, Ord point, Ord r, Num r) => set point -> Triangulation point Source #

Naive \( O(n^4) \) time implementation of the delaunay triangulation. Simply tries each triple (p,q,r) and tests if it is delaunay, i.e. if there are no other points in the circle defined by p, q, and r.

pre: the input is a *SET*, i.e. contains no duplicate points.

toPlaneGraph :: forall {k} (s :: k) point r. (Point_ point 2 r, Ord r, Num r) => Triangulation point -> CPlaneGraph s point () () Source #

ST' is a strict triple (m,a,x) containing:

  • m: a Map, mapping edges, represented by a pair of vertexId's (u,v) with u < v, to arcId's.
  • a: the next available unused arcID
  • x: the data value we are interested in computing type ST' a = ST (SM.Map (VertexID,VertexID) ArcID) ArcID a

convert the triangulation into a plane graph

running time: \(O(n)\).
