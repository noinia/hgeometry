stab :: (Ord r, ClosedInterval_ interval r) => r -> SegmentTree f interval -> [f interval] Source #

Query the segment tree, returns a list of canonical subsets; one for each relevant node. Each element of the resulting list represents all intervals stored at that node that are stabbed by q.

\(O(\log n)\)
