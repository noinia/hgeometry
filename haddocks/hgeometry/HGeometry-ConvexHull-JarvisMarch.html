convexHull :: (Ord r, Num r, Point_ point 2 r) => NonEmpty point -> ConvexPolygon point Source #

Compute the convexhull using JarvisMarch.

running time: \(O(nh)\), where \(n\) is the number of input points and \(h\) is the complexity of the hull.

upperHull :: (Num r, Ord r, Point_ point 2 r) => NonEmpty point -> NonEmpty point Source #

Computes the upper hull. The points are reported in left-to-right order.

running time: \(O(nh)\), where \(h\) is the complexity of the upper hull.

upperHull' :: (Num r, Ord r, Point_ point 2 r) => NonEmpty point -> NonEmpty point Source #

Upepr hull from left to right, without any vertical segments.

lowerHull :: (Num r, Ord r, Point_ point 2 r) => NonEmpty point -> NonEmpty point Source #

Computes the lower hull, from left to right. Includes vertical segments at the start.

running time: \(O(nh)\), where \(h\) is the complexity of the hull.
