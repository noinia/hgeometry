samplePolygon :: (SimplePolygon_ polygon point r, Ord r, Real r, UniformRange r, StatefulGen g m) => polygon -> g -> m (Point 2 Double) Source #

Uniformly samples a point in a polygon.

\(O(n\log n)\)

samplePolygons :: (SimplePolygon_ polygon point r, StatefulGen g m, Foldable1 nonEmpty, Real r, Ord r, UniformRange r) => nonEmpty polygon -> g -> m (Point 2 Double) Source #

Unfiormly samples a point from a set of polygons. You may want to build a pointSampler if the goal is to sample multiple points.

\(O(N\log N)\), where \(N\) is the total size of all polygons.
