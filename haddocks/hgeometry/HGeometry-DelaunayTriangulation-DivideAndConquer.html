Divide & Conqueror Delaunay Triangulation

delaunayTriangulation :: (Foldable1 set, Point_ point 2 r, Ord point, Ord r, Num r) => set point -> Triangulation point Source #

Computes the delaunay triangulation of a set of points.

Running time: \(O(n \log^2 n)\) (note: the extra \(\log n\) factor is since we use an IntMap in the implementation)

pre: - the input is a *SET*, i.e. contains no duplicate points. - no three colinear points - no four cocircular points
