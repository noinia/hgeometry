Given a polygon, and candidate visibility edges (edges guaranteed not to intersect) the boundary; test if the edges are actually inside the polygon (as well as adding the actual polygon edges as visible).

\(O(n + m)\), where m is the number of candidate edges.

data DefinerF r edge orig Source #

A vertex of a visibility polygon; which is either an original vertex of type orig or a vertex that is the intersection point of a ray through an original vertex and an edge.
